O que é HTML, CSS e JavaScript?

HTML

Linguagem de marcação usada para estruturar conteúdo na web.

<h1>Olá, mundo!</h1>

Use para estruturar textos, imagens, links e tabelas.

CSS

Linguagem de estilos que define a aparência do HTML.

h1 { color: blue; }

Use para cores, fontes, espaçamentos e layouts.

JavaScript

Linguagem de programação que dá interatividade às páginas.

alert("Olá, mundo!");

Use para criar animações, validações e aplicativos dinâmicos.
